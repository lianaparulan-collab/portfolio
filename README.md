# tarot-portfolio
Here is a humble tarot reading website made by me!
Built with html, css, and javascript

My project includes tarot card data from the tarot-json project:
https://github.com/metabismuth/tarot-json

Licensed under the MIT License.
I used this merely for the images of the cards. 

Backgrounds and other images were sourced primarily from https://unsplash.com/

Use of AI:

Card meanings (upright and reversed) were created with the assistance of ChatGPT and further refined by me to reflect my own interpretations and style.
This was not included in the referenced repository.
- see tarot-meanings.json

AI tools, including Claude and ChatGPT, were used to assist in the development of CSS animations, which were subsequently adapted and implemented by me.
